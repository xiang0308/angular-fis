### ry-min-max

* 给数字输入框限制

#### config配置

| 属性       | 类型           | 描述         |
| ---------- | -------------- | ------------ |
| ry-min-max | array          |              |
| -> 0       | number or null | 最小值       |
| -> 1       | number or null | 最大值       |
| -> 2       | number or null | 允许输入小数 |
|            |                |              |

#### code

```html
<input type="text" ry-min-max="[0, 100, 2]" />
```
